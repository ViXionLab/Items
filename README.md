## シースルーカバー 交換方法 

交換の前に本手順をよくお読みいただいた上で手順通りに交換をお願いします。 

シースルーカバー取り外し時には無理に力を掛けすぎないようにご注意ください。 

(ツメが折れてしまうことがあります) 

 
 

シースルカバーとは本体の ④ の部分を指します。 

 

 

### 準備していただくもの 

 

精密ドライバー (+) 

 

 

 


### シースルーカバーの取り外し方法 

 

1. 本体を裏返し、写真の箇所のネジを精密ドライバーで緩めて取り外します。 
（取り外すネジは１箇所のみです） 

 

 

2. カバーをスライドさせ持ち上げます。 

 

3. 写真のようにゆっくり外してください。 

 

 

 

 

### シースルーカバーの取り付け方法 

　取り外し方法と逆の手順となります。 

1. シースルーカバーの二箇所のツメを溝に嵌め込みます。 

 
2. ゆっくり装着します。 

 
3. 上部二箇所のツメを差し込みます。 


4. 写真の箇所のツメを嵌め込みます。 

5. シースルーカバーのツメが全て本体に装着されていることを確認し、精密ドライバーでネジを締めます。 

 

 

 

 

## 鼻パッド 交換方法 

 

 

### 準備していただくもの 

精密ドライバー (+) 

 

 

 

### 手順 

 
写真の2箇所のネジを精密ドライバーで緩めて取り外します。 


鼻パッドを取り外し交換します。
